# Photos produit — SAVO-AGRI

Déposez ici les deux photos du produit, avec **exactement** ces noms de fichier :

| Fichier attendu              | Produit            | Image fournie                |
| ---------------------------- | ------------------ | ---------------------------- |
| `savo-agri-5l.jpg`           | SAVO-AGRI **5 L**  | le bidon (grande image)      |
| `savo-agri-1l.jpg`           | SAVO-AGRI **1 L**  | la bouteille (petite image)  |

## Comment faire

1. Enregistrez les deux images envoyées sur votre ordinateur.
2. Renommez-les `savo-agri-5l.jpg` et `savo-agri-1l.jpg`.
3. Glissez-les dans ce dossier : `public/products/`.

La section **Produits** du site les chargera automatiquement
(`/products/savo-agri-5l.jpg` et `/products/savo-agri-1l.jpg`).

> Tant que les fichiers ne sont pas présents, un dégradé vert s'affiche en repli —
> la mise en page reste donc propre.

Formats acceptés : `.jpg`, `.png` (gardez l'extension `.jpg` dans les noms ci-dessus,
ou prévenez-moi si vous utilisez `.png` afin que je mette à jour les chemins).
